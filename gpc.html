Project Description

During my internship at Global Power Consulting (GPC), I developed a photovoltaic data simulator. This web-based simulator enables users to visualize and select the best solar panel installations for specific locations by leveraging APIs to provide real-time data on energy efficiency and costs. Key features include:

  • Interactive map integration
  • Real-time energy data visualization
  • Customized recommendations for photovoltaic installations

Results

The project provided users with an efficient tool to assess and compare solar panel options, reducing decision-making time for photovoltaic installations. This internship strengthened my API integration and front-end development skills.
